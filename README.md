# Netflix Clone
A Netflix clone built using React, Firebase, and TMDB API.

## Features
- User authentication (via Firebase)
- Fetch and display movies from TMDB API
- Responsive design
- Dynamic categories for movies/shows

## Tech Stack
- React
- Firebase
- CSS/SCSS
